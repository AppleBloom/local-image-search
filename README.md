Description
===========
Local image search for Gnome{2,3} utilizing Tracker semantic engine.

Usage
=====
Run `search by tag/search_by_tag.py`.

Example query
-------------
`("sailor.moon" | SM) & .Mars`
-- finds images that have both:
* `sailor` separated by one symbol from `moon` in their path or tag `SM`
* `Mars` in their path
